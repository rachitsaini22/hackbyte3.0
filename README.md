# 💻 Codium

Codium is a real-time collaborative 1v1 coding platform that allows users to code together, interview candidates, or practice pair programming — all in the browser.



---

## 🚀 Features

- 👨‍💻 **1v1 Collaboration**: Code simultaneously with another user in real-time
- 📝 **Multiple Language Support**: Write and run code in popular programming languages
- 📡 **Low Latency Syncing**: Smooth live code synchronization using websockets
- 💬 **Chat Support**: Built-in chat to communicate while coding
- 🔐 **Secure Rooms**: Unique room IDs for private sessions

---

## 🛠️ Tech Stack

- **Frontend**: React.js, Tailwind CSS
- **Backend**: Node.js, Express
- **Realtime Engine**: Socket.IO
- **Code Execution**: Dockerized language environments / Judge0 API *(or your choice)*
- **Database**: MongoDB / PostgreSQL *(if needed)*
